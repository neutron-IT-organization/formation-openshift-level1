## Déroulé recommandé

1. Finir le chapitre théorique
2. Ouvrir le manager Rahoot, sélectionner le bon quiz
3. Afficher `https://rahoot.apps.neutron-sno-office.neutron-it.fr` au vidéoprojecteur
4. Demander aux participants de se connecter avec le code affiché
5. Lancer le quiz quand tout le monde est prêt
6. Après chaque question, commenter la bonne réponse
